*Monod*, a package for CME inference from scRNA-seq data
===================================

*Monod* implements procedures for fitting and analyzing stochastic models of transcription and sequencing.

The :doc:`usage` page documents how to use the software, including its :ref:`installation`.

For an example of the software applied to a small dataset, see the `demo Colaboratory notebook <https://colab.research.google.com/github/pachterlab/monod/blob/anndata/Monod_demo_new.ipynb>`_.

.. note::

   This project and its documentation are under active development.
